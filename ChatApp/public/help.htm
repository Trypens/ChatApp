Commands

/me text

background announcement in chat, bold but colorless, with /me replaced by your username

Example: Homo Trypens> /me writes chat help 00:00:00 Homo Trypens writes chat help
/pass username text /pass "user name" text

pass another user something in the chat background.

Example: Homo Trypens> /pass "The Traveler" the pipe 00:00:00 Homo Trypens passes the pipe to The Traveler
